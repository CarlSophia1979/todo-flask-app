# To-Do Flask App

A simple to-do list web app built with **Python Flask**, **SQLite**, and **HTML/CSS**.

---

## 🔧 Features

- Add and delete tasks  
- Tasks are saved using a local SQLite database  
- Styled with basic CSS  
- Easy to run locally
